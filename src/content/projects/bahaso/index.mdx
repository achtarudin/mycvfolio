#### 🌐 Bahaso Platform

Bahaso adalah platform pembelajaran bahasa berbasis teknologi yang menyediakan berbagai program belajar seperti **TOEFL Prediction Test**, **Placement Test**, dan **Kelas Bahasa Inggris** secara daring.

Didirikan sejak 2015, Bahaso telah berkembang menjadi platform edutech dengan sistem pembayaran mandiri, pengelolaan sertifikat otomatis, dan dashboard belajar untuk pengguna.

---

#### ✨ Fitur Utama

- 🏠 **Landing Page**
  - Desain modern dan responsif
  - Ajakan bergabung dan penjelasan produk
- 🛒 **Pembelian Course**
  - Pembayaran langsung via virtual account / kartu kredit
  - Penjadwalan tes fleksibel dan otomatis
- 📊 **Dashboard Pengguna**
  - Menampilkan progres belajar & skor tes
  - Integrasi sertifikat otomatis
- 🧾 **Riwayat Transaksi**
  - Menyimpan histori pembelian
  - Fitur "lanjut pembayaran" dengan batas waktu
- 📜 **Sertifikat**
  - Dapat diunduh setelah menyelesaikan tes
  - Validasi dan tampilan profesional

---

#### 🛠️ Teknologi yang Digunakan

- **Frontend:** Vue.js (Nuxtjs.js) + Prisma.js
- **Backend:** Laravel REST API + LMS Moodle
- **Database:** MySQL
- **Development:** Docker, Git
- **Pembayaran:** Midtrans (Virtual Account, Credit Card, QRIS)
- **Sertifikat:** PDF Generator (Server-side)
