#### 🧾 Deskripsi Project

**Untar Dashboard** adalah sistem **backoffice internal** yang dikembangkan untuk **Universitas Tarumanagara (UNTAR)** dalam rangka mendukung kegiatan pelatihan online berbasis sertifikat. Sistem ini memungkinkan admin untuk:

- Mengelola ujian dan peserta
- Memonitor aktivitas ujian
- Meng-generate sertifikat secara otomatis berdasarkan periode dan grup

Project ini ditujukan untuk mempermudah pelaksanaan ujian daring dan penerbitan sertifikat melalui sistem terintegrasi.

---

#### 🔧 Fitur Utama

- 📝 **Generate Sertifikat**
  - Filter berdasarkan tanggal dan grup
  - Pembuatan data sertifikat otomatis dari hasil ujian
  - Status validasi dan error tracking

- 📊 **Monitoring Ujian**
  - Statistik aktivitas pengguna selama ujian
  - Status keaktifan peserta dan progress kursus
  - Visualisasi data berdasarkan periode tertentu

---

#### 🧑‍💻 Teknologi yang Digunakan

- **Frontend:** Laravel + Livewire
- **Backend:** Laravel REST API + LMS Moodle
- **Database:** Mysql, Redis
- **Development:** Docker, Git
- **PDF Generator:** HTML → PDF backend rendering
